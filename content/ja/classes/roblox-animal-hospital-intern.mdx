- **第1診察室から第5診察室**での初期の患者治療手順。
- 最初の3つのシフトにおける**正気度メーター**の管理。

<Section icon={ClipboardCheck} title="コアゲームプレイループとシフトの生存">

**Roblox 動物病院インターン**として、あなたのシフトは「受付」と「治療」という2つの緊迫したフェーズに分かれます。クリニックに入る動物が正当な患者であり、危険な異変ではないことを確認しなければなりません。異変を受け入れてしまうと正気度が低下し、病院の廊下でプレイヤーや患者を積極的に狩るスキンウォーカーが出現する可能性があります。

<Steps>
  <Step num="1" title="初期の受付チェック">
    患者がデスクに近づいたら、視覚スキャンを行います。余分な目、リアルな人間のような歯、奇妙な痙攣など、明らかな奇形がないか確認します。少しでも「おかしい」と感じたら、入れてはいけません。
  </Step>
  <Step num="2" title="写真とCCTVの確認">
    目の前の患者と提供された写真を比較します。写真が異なる種を示していたり、「呪われた」要素（血走った目など）がある場合は、すぐにデスクのシャッターを使用してください。CCTVをチェックして、患者の影や反射に隠れた姿が現れていないか確認します。
  </Step>
  <Step num="3" title="診察室の割り当て">
    安全が確認されたら、患者を開いている診察室に送ります。各診察室（1〜8番）には特定の診断要件があります。コンピュータ端末を使用して病気を特定し、必要な治療アイテムを確認してください。
  </Step>
  <Step num="4" title="緊急対応">
    診察室の火災や「ベッドモンスター」などのランダムなイベントに備えてください。施設と残りの正気度を守るために、必要に応じて消火器やテーザー銃を使用します。
  </Step>
</Steps>

<Callout type="warning" title="正気度のリスク">
「呪われた」写真を3秒以上見つめると、正気度が急速に低下します。異変を特定したら、すぐに目をそらすかシャッターを閉めてください。
</Callout>

</Section>

<Section icon={Eye} title="異変の検出：脅威の特定">

**Roblox 動物病院インターン**にとって最も重要なスキルは、異変の検出です。病院は常に、病棟に潜入しようとする実体からの脅威にさらされています。これらの異変はしばしば通常の動物患者を模倣しますが、フロントデスクを通過する前に見抜かなければならない微妙な手がかりを残します。

| 検出方法 | 正常な兆候 | 異変の警告（即座に拒否） |
| :--- | :--- | :--- |
| **視覚スキャン** | 2つの目、標準的な毛並み | **3つの目、不自然に広い笑顔、痙攣する肢体** |
| **CCTVモニター** | 正常な動き | **伸びた手足、カメラを凝視、体が黒い空洞** |
| **患者の写真** | 患者と一致 | **呪われた笑み、血走った目、異なる種類** |
| **行動** | 落ち着いて待機 | **攻撃的な発言、突然の消失、浮遊** |

<CardGrid cols={2}>
  <Card title="スキンウォーカーの脅威">
    - **原因**: 異変を見逃して受け入れてしまう。
    - **危険性**: 高い正気度低下と物理攻撃。
    - **対策**: **テーザー銃**や**銃**を使用して無力化する。
  </Card>
  <Card title="ベッドモンスター">
    - **原因**: 治療用ベッドの下にランダムに出現。
    - **危険性**: 治療を妨害し、プレイヤーを驚かせる。
    - **対策**: 懐中電灯や特定のアイテムを使用して追い払う。
  </Card>
</CardGrid>

<Callout type="info" title="環境的な危険">
すべての脅威が患者であるとは限りません。天井に潜む「シーリング・リーチ」の異変に注意し、不必要な正気度の損失を防ぐために床の緑色のスライムを避けてください。
</Callout>

<Section icon={Package} title="必須アイテムと治療ツール">

**Roblox 動物病院インターン**にとって、インベントリの管理は非常に重要です。インターンは看護師クラスのような拡張インベントリを持っていないため、持ち運ぶアイテムを厳選する必要があります。正気度を回復させるアイテムや、環境の脅威から身を守るアイテムを常に優先してください。

| アイテム名 | カテゴリ | 主な用途 | 優先度 |
| :--- | :--- | :--- | :--- |
| **コーヒー** | 正気度 | 正気度を20〜30ポイント回復 | **最優先** |
| **テーザー銃** | 防御 | スキンウォーカーや敵対的な患者をスタンさせる | **高** |
| **消火器** | 緊急 | 患者を救うために診察室の火を消す | **中** |
| **薬草** | 治療 | シフト初期の患者に対する基本的な治療 | **高** |
| **手術道具** | 専門 | 第8診察室での処置に必要 | **状況次第** |

<Callout type="success" title="インベントリ管理">
チームでプレイしている場合は、インターンがフロントデスクのチェックを担当している間、一人のプレイヤーにショップからコーヒーや正気度回復アイテムを持ってくることに専念してもらうと良いでしょう。
</Callout>
