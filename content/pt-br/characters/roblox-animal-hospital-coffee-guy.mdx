### As Melhores Classes para Gerenciamento de Sanidade

- **Cirurgião (Tier-S)**: Esta é a melhor classe gratuita para jogar. Ele ganha Sanidade e um aumento de velocidade toda vez que você cura um paciente com sucesso. No Nível 3, o ganho de Sanidade é duplicado, muitas vezes eliminando completamente a necessidade de café se você for rápido nos tratamentos.
- **Psicólogo (Tier-A)**: Uma classe especializada que duplica todos os efeitos de Sanidade. Isso significa que o café restaura o dobro, mas os erros também machucam o dobro. Recomendado para jogadores experientes que raramente perdem uma anomalia.
- **Secretária (Tier-A)**: A classe perfeita para a recepção. Você recupera Sanidade apenas fazendo o check-in correto dos pacientes. Se você for o **roblox animal hospital coffee guy** na recepção, esta classe é uma compra obrigatória por 120 Moedas de Animal.
- **Estagiário (Tier-C)**: A classe inicial. Ela fornece um bônus fixo para sua Sanidade máxima (até +20 no Nível 3), tornando-se uma escolha segura para iniciantes que estão aprendendo as mecânicas.
